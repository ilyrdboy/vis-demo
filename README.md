# 都城韧性：北京历史转折期的冲击、恢复与功能重组

这是一个面向 ChinaVis 2025 数据可视化竞赛赛道 1-I「析城观史」的 Vue 可视分析原型。项目围绕北京城市历史演化中的关键转折期，组织多源 Excel 数据，展示历史冲击、人口迁移、制度重组、交通水系与城市功能变化之间的关系。

## 数据来源

项目读取 `D:\vis` 根目录下的赛题 Excel 数据，包括水系、气候、植被、灾害、建制沿革、重点建筑、其他建筑、人口、文化、商业手工业、物产、交通、事件、战争、人物等 15 类数据。

数据生成脚本会输出：

- 每个数据集的行数、列数、文件大小、工作表列表
- 每个数据集的朝代分布
- 每个数据集的前 10 个关键词
- 每个数据集的可追溯代表记录
- 面向历史转折点的真实证据索引

每条代表记录包含：

- `dataset`
- `filename`
- `sheet`
- `row`
- `period`
- `title`
- `rawText`
- `source`
- `keywords`

## 运行方式

安装依赖后启动开发服务器：

```powershell
cd D:\vis\vis-demo
npm install
npm run dev
```

生产构建：

```powershell
cd D:\vis\vis-demo
npm run build
```

## 生成前端数据

当前前端使用的数据文件为：

```text
src/data/processedData.js
```

重新从 Excel 生成：

```powershell
cd D:\vis\vis-demo
python tools\build_data.py
```

如果系统 Python 没有 `openpyxl`，可以使用 Codex 工作区自带 Python，或安装依赖：

```powershell
pip install openpyxl
```

## 当前功能

- 历史转折时间线：选择燕云入辽、金迁都、蒙古攻占中都、明军攻占大都、新中国定都北京等节点。
- 转折机制图：用因果链表达政权、人口、空间、水系、交通和城市功能的关系。
- 真实证据面板：根据选中的转折点，从相关数据集里展示可追溯原始记录。
- 数据图谱：展示 15 类数据的规模与朝代分布缩略图。
- 数据集详情面板：点击任一数据集后查看朝代分布、关键词、样例原文、来源文件、工作表和数据质量信息。
- 朝代密度矩阵：比较核心数据集在不同时期的记录密度。
- 机制分析板：从外部冲击、内部重组、空间支撑三个角度组织分析入口。
